Module mondo-bank

Usage

mondo = require('mondo-bank')

All methods return a promise but can optionally be called with a callback function as the final argument

Promise style

methodPromise = mondo[$method]([$params])
methodPromise
   .then(function(value){
     ...
   })
   .catch(function(err){
     ...
   })

Callback style

mondo[method]([$params], function(err, value){
  if (err) {
   ...
  }
  ...
})
